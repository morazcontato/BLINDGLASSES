# MANUAL DE MARCA — BASE PARA LANDING PAGES

## OBJETIVO
Criar uma identidade moderna, limpa e focada em conversao.

---

## PALETA DE CORES

- Branco: #FFFFFF (fundo principal)
- Preto: #000000 (textos e contraste)
- Azul Claro: #4DA6FF (CTAs e interacoes)

REGRAS:
- Maximo 3 cores
- Azul = acao
- Evitar poluicao visual

---

## TIPOGRAFIA

Titulos:
- Inter
- Poppins
- Montserrat

Textos:
- Inter
- Open Sans
- Roboto

REGRAS:
- Max 2 fontes
- Titulos: 600-700
- Texto: 400-500
- Priorizar legibilidade

---

## ESTILO VISUAL

- Minimalista
- Clean
- Espacamento amplo
- UX-first

COMPONENTES:

Botoes:
- Fundo azul (#4DA6FF)
- Texto branco
- Border-radius: 6-10px
- Hover: azul mais escuro

Cards:
- Fundo branco
- Sombra leve
- Borda arredondada

Secoes:
- Branco + cinza leve (#F5F5F5)

---

## ESTRUTURA DE LANDING PAGE

1. Hero

---

## TOM DE VOZ

- Direto
- Objetivo
- Confiante
- Sem exagero

---

## NAO FAZER

- Nao usar muitas cores
- Nao usar fontes decorativas
- Nao poluir layout
- Nao textos longos

---

## UX REGRAS

- Mobile-first
- Hierarquia clara
- Botoes visiveis
- Texto escaneavel

---

## INSTRUCAO PARA CLAUDE

Sempre que gerar landing page:
- Seguir este manual
- Usar cores definidas
- Manter layout clean
- Priorizar conversao
